Pext is your everything

Pext stands for Python-based extendable tool. It provides a reasonably minimal core and the ability to load modules, making it simple yet powerful. As long as it can be expressed in a filterable list and possibly with command-line style commands, it can be expressed in Pext.

Currently, modules exist for password management, todo-list management, emoji selection, quickly connecting to a SSH server and weather information retrieval. However, with the power of Python, many more plugins can easily be created.
